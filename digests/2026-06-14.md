# 🧠 Personal Development

## Daniel Pink

**[Why Finishing Feels Worse Than Starting](https://www.youtube.com/watch?v=2zyIZV-M8mc)**
The thesis: the final stretch of a boring task feels the worst — not because the task changed, but because your brain notices how much you've already endured. Zoom out, and the task shrinks.

- **Effort already endured distorts the finish line.** The brain registers what it has already been through, which makes an unchanged workload feel heavier near the end. The task hasn't changed; what the brain is tracking has.
- **Zooming out resets the scale.** Stepping back so the full task is visible makes the remaining portion shrink relative to the whole — a direct counter to the amplified discomfort of the final stretch.

## Databricks Blog

**[Introducing Omnigent: A Meta-Harness to Combine, Control and Share Your Agents](https://www.databricks.com/blog/introducing-omnigent-meta-harness-combine-control-and-share-your-agents)**
Databricks released Omnigent, an open-source (Apache 2.0) meta-harness that sits above individual agent harnesses to make them interoperable — built from internal experience using and building agents at scale for coding workflows.

- **Coordination above agents, not replacement of them.** Omnigent adds a layer above harnesses like Claude Code, Codex, and custom agents so they can be composed and switched with minimal code changes, without rewriting each harness.
- **Stateful policies replace prompt-level guardrails.** Cost budgets and permission guardrails are enforced at the meta-harness layer rather than through prompts — making controls persistent and auditable rather than advisory.
- **Live session sharing as a first-class feature.** Teams can share a running agent session via URL to review, comment, and steer agents together in real time — targeting the copy-paste-between-tabs workflow that currently bridges agent outputs and collaboration tools.
- **Cloud execution and sandboxing built in.** Omnigent supports cloud execution on services like Modal and Daytona, with OS-level sandboxing and network request interception for safer agent runs.
