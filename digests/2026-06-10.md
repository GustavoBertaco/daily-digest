## AWS What's New

**[OpenAI GPT-5.4 and GPT-5.5 now on Amazon Bedrock (US East)](https://aws.amazon.com/about-aws/whats-new/2026/06/openai-gpt-us-east-virginia-amazon/)**
GPT-5.4 and GPT-5.5 expanded to US East (N. Virginia) on Bedrock. GPT-5.5 is OpenAI's most capable model, targeting advanced coding, research, document workflows, and long-running agentic tasks.

**[Amazon ECS Managed Daemons: inter-task visibility](https://aws.amazon.com/about-aws/whats-new/2026/06/ecs-managed-daemons-pid-ipc-modes/)**
Two new ECS daemon settings: `pidMode` (daemon sees all host processes) and `ipcMode` (shared IPC namespace). Unblocks tracing, profiling, and security agents that need access to application processes.

**[Amazon EC2 M9g and M9gd — Graviton5 GA](https://aws.amazon.com/about-aws/whats-new/2026/06/ec2-m9g-m9gd-instances-graviton5-processors-available)**
Fifth-gen Graviton-powered general purpose instances are now GA. ClickHouse reported 36% performance boost over M8g with zero code changes; Honeycomb achieved 36% better throughput per core.

**[EC2 P6-B200 now in AWS GovCloud (US-East)](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-p6-b200-aws-govcloud/)**
NVIDIA Blackwell GPU instances reach GovCloud. Up to 2× performance vs P5en, 8 Blackwell GPUs with 1440 GB HBM, 3.2 Tbps networking.

**[CUR 2.0: table configuration updates via Console and SDK](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-cost-usage-report/)**
AWS Cost and Usage Report 2.0 now supports modifying existing exports (granularity, columns, format, destination) without deleting and recreating them.

# 🤖 AI & Machine Learning

## AI Engineer Conference

**[Stop Making Models Bigger, Make Them Behave — Kobie Crawford, Snorkel](https://www.youtube.com/watch?v=TNwJ1LMiENk)**
Snorkel's research: the biggest gains often come from better data, not larger models. Covers a specific use case in partnership with UC Berkeley's RLLM group — high-quality expert-in-the-loop data applied to the right problem delivers outsized results vs. scaling parameters.

**[Sovereign Escape Velocity: Ownership with Open Models — Google DeepMind](https://www.youtube.com/watch?v=SS-A8sE7hkw)**
Google DeepMind presents Gemma 4: four model sizes, two targeting mobile/IoT (E2B/E4B). The core thesis — Gemini is the most capable hosted option, but when data can't leave your infrastructure, open models are the only path. Strong framing for the proprietary vs. open-source decision.

**[Self-Driving Products: Product Signals to Pull Requests — Joshua Snyder, PostHog](https://www.youtube.com/watch?v=zMiSRliEzv4)**
PostHog is building a pipeline where product signals (errors, metric changes) trigger background agents that investigate root causes and open PRs automatically — skipping the dashboard entirely. Lessons: signal ingestion, agent orchestration, and safe auto-deployment behind feature flags.
